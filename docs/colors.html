Colors

Convey meaning through color.

Out of the box you get access to all colors in the Sonnat Design System. You can use any color anywhere, all colors change with the theme.

import { useTheme } from '@sonnat/ui/styles'


const Component = () => {
  const { colors, swatches } = useTheme()


  ...
}
import { makeStyles } from "@sonnat/ui/styles";


const useStyles = makeStyles(
  ({ colors, swatches }) => ({
    root: {
      ...
    },
    ...
  }),
  { name: "MyComponentStyle" }
);
To learn more about how to create and customize theme, check out the createTheme's documentation.

Primary Color

Represents primary interface elements for a user. It's the color displayed most frequently across your app's screens and components.

lightertheme.colors.primary.light#EE3F7C
origintheme.colors.primary.origin#EC135F
darkertheme.colors.primary.dark#C0114E

Secondary Color

Represents secondary interface elements for a user. It provides more ways to accent and distinguish your product. Having it is optional.

lightertheme.colors.secondary.light#6D82EE
origintheme.colors.secondary.origin#1337EC
darkertheme.colors.secondary.dark#112EC0

Status Colors

Represents a clear message as well as status, such as success, error, failure, reminder, link and so on. Status color selection need to comply with the user's basic understanding of color.

Error

Indicates errors in components, such as invalid text in a text field.

lightertheme.colors.error.light#EE3F3F
origintheme.colors.error.origin#EC1313
darkertheme.colors.error.dark#C01111

Success

Indicates the successful completion of an action.

lightertheme.colors.success.light#11C031
origintheme.colors.success.origin#119229
darkertheme.colors.success.dark#106520

Warning

Represents potentially dangerous actions or important messages.

lightertheme.colors.warning.light#EE853F
origintheme.colors.warning.origin#EC6A13
darkertheme.colors.warning.dark#C05711

Info

Presents information to the user that is neutral and not necessarily important.

lightertheme.colors.info.light#1387EC
origintheme.colors.info.origin#116FC0
darkertheme.colors.info.dark#115692
