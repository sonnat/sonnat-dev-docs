Badge

Displays an indicator that requires attention.

Filled Badges

Filled badges show text or number, using primary and secondary colors.
You can use them as a standalone badge or as a relative badge applied to its children.

Dotted Badges

Dotted badge is a small dot indicates that something has changed without giving a count.
Just like the filled badges, you can have dotted badges as standalone or relative to its child.

This is a typography!

<Badge variant="dotted" />
<Badge color="secondary" variant="dotted" />
<Badge variant="dotted">
  <Text variant="caption" as="p">
    This is a typography!
  </Text>
</Badge>
<Badge variant="dotted" color="secondary">
  <IconButton
    aria-label="Demo download button"
    icon={<Download />}
    variant="inlined"
  />
</Badge>

To have the larger or smaller dotted badges, use the dotSize property.

<Badge dotSize="small" variant="dotted" />
<Badge dotSize="medium" variant="dotted" />
<Badge dotSize="large" variant="dotted" />

Better Placement

You can use the childShape property to place the badge way more accurately relative to the corner of the wrapped child(ren).

<Badge variant="dotted" childShape="rectangular">
  // Rectangular shape
  <div></div>
</Badge>
<Badge variant="dotted" childShape="circular">
  // Circular shape
  <div></div>
</Badge>

Visibility

The visibility of badges can be controlled using the visible property.

import { Badge, Button } from "@sonnat/ui";
import * as React from "react";


const BadgeVisibilityDemo: React.FC = () => {
  const [isVisible, setVisible] = React.useState(true);


  return (
    <React.Fragment>
      <Badge
        visible={isVisible}
        variant="dotted"
        color="secondary"
      >
        // Rectangular shape
        <div></div>
      </Badge>
      <Button
        size="small"
        variant="outlined"
        label="Toggle visibility"
        onClick={() => void setVisible(v => !v)}
      />
    </React.Fragment>
  );
};


export default BadgeVisibilityDemo;

Alignment

You can use horizontalPosition and verticalPosition properties to move the badge to any corner of the wrapped element.

9
<Badge
  variant="dotted"
  color="secondary"
  verticalPosition="top"
  horizontalPosition="right"
>
  // Rectangular shape
  <div></div>
</Badge>
<Badge
  childShape="circular"
  color="secondary"
  text="9"
  verticalPosition="top"
  horizontalPosition="right"
>
  // Circular shape
  <div></div>
</Badge>
